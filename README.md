## ❗ Troubleshooting

| ❌ Problem | ✅ Fix |
|---|---|
| Toolkit not detecting Blender | Ensure Blender 3.6+ is installed, verify path |
| UV unwrap distorted | Increase island margin, check for overlapping faces |
| Export fails | Check write permissions on output folder |
| Viewport still slow | Enable proxy display mode in Viewport Booster |
| Materials missing after batch | Run Material Batcher in preview mode first |
| Normals still flipped | Use "Recalculate Outside" before Normal Corrector |
